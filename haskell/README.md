install haskell:
```
apt-get install haskell-platform -y
haskell-compiler --version
```
use haskell in terminal:
```
echo ":set prompt \"ghci> \"" > ~/.ghci
ghci
```
compile haskell program:
```
ghc -o example example.hs
```
